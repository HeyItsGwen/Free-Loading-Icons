About

All icons on this page are made with HTML and CSS. No frameworks required!

Simply click the 'Get The Code' button, copy/past the HTML and CSS into your project, and you're done.

If you'd like to customize the icons, there are instructions in the CSS that tell you what to change to achieve certain effects.

Visual Based

Click Me to Close

                    HTML<>
                    
                    <div id='visualOne'>
                      <div class='visualOne'></div>
                      <div class='visualOne'></div>
                      <div class='visualOne'></div>
                      <div class='visualOne'></div>
                      <div class='visualOne'></div>
                    </div>
    
                    CSS{}
    
                    #visualOne {
                      display: flex;
                      flex-direction: row;
                      /* HEIGHT OF THE CONTAINER FOR THE BARS*/
                      height: 30px;
                      align-items: center;
                    }                
                    .visualOne {
                      /* STARTING HEIGHT AND WIDTH OF THE BARS */
                      height: 30px;
                      width: 6px;
                      /* YOUR PREFERED BORDER COLOR/SIZE */
                      border: 1px solid rgb(73, 73, 73);
                      /* SPACE BETWEEN EACH BAR */
                      margin: 0 1px;
                      /* BACKGROUND COLOR FOR THE BARS */
                      background-color: white;
                    }
                    /* '3s' AFTER bouncingBars IS LENGTH 
                    OF THE ENTIRE ANIMATION */
                    /* TIME AFTER ease-in-out IS THE DELAY 
                    FOR EACH BLOCK'S ANIMATION */
                    .visualOne:first-child {
                      animation: bouncingBars 3s ease-in-out infinite;
                    }                
                    .visualOne:nth-child(2) {
                      animation: bouncingBars 3s ease-in-out 0.3s infinite;
                    }                
                    .visualOne:nth-child(3) {
                      animation: bouncingBars 3s ease-in-out 0.6s infinite;
                    }                
                    .visualOne:nth-child(4) {
                      animation: bouncingBars 3s ease-in-out 0.9s infinite;
                    }                
                    .visualOne:last-child {
                      animation: bouncingBars 3s ease-in-out 1.2s infinite;
                    }                
                    @keyframes bouncingBars {
                      25% {
                        /* HEIGHT YOU WANT THE BARS TO CHANGE TO*/
                        height: 15px;
                      } 50% {
                        /* SAME AS ORIGINAL HEIGHT IN .visualOne */
                        height: 30px;
                      }
                    }

Click Me to Close

                  HTML<>
    
                  <div id='iconOne'>
                    <p>
                      <span class='loading'>Loading</span>
                      <span class='loadingDotOne'>.</span>
                      <span class='loadingDotTwo'>.</span>
                      <span class='loadingDotThree'>.</span>
                    </p>
                  </div>
    
                  CSS{}
      
                  #iconOne {
                    /* COLOR OF TEXT/DOTS*/
                    color: rgb(73, 73, 73);
                    /* WEIGHT OF THE TEXT/DOTS */
                    font-weight: bold; 
                  }    
                  /* '2s' AFTER iconOneLoading IS LENGTH 
                  OF THE ENTIRE ANIMATION */
                  /* TIME AFTER ease-in-out IS THE DELAY 
                  FOR EACH BLOCK'S ANIMATION */          
                  #iconOne .loading {
                    animation: iconOneLoading 2s ease-in-out infinite;
                  }              
                  #iconOne .loadingDotOne {
                    animation: iconOneDots 2s ease-in 0.2s infinite;
                  }              
                  #iconOne .loadingDotTwo {
                    animation: iconOneDots 2s ease-in 0.8s infinite;
                  }              
                  #iconOne .loadingDotThree {
                    animation: iconOneDots 2s ease-in 1.3s infinite; 
                  }              
                  @keyframes  iconOneDots {
                    30% {
                      /* BACKGROUND COLOR */
                      color: rgb(225, 255, 255);
                    }
                    50% {
                      /* WANTED FONT COLOR*/
                      color: rgb(127, 146, 146);
                    }
                  }              
                  @keyframes iconOneLoading {
                    10% {
                      /* LIGHTER VERSION OF YOUR FONT COLOR */
                      color: rgb(136, 136, 136);
                    }
                    60% {
                      /* YOUR ORIGINAL FONT COLOR */
                      color: rgb(73, 73, 73);
                    }
                  }
